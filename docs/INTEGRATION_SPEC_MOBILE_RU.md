# Credit Platform v5: Интеграция mobile/BFF

Этот файл больше не является основным источником интеграционного контракта.

Актуальный документ:

- [INTEGRATION_SPEC_IT_RU.md](./INTEGRATION_SPEC_IT_RU.md)

Использовать его нужно для:

- mobile BFF;
- внешней IT-системы;
- любого канала, который передает в платформу профиль заявителя в формате `Applicant Input v2`.

Кратко:

- внешний канал передает flat applicant profile;
- `request_id` генерирует платформа;
- routing не задается клиентом;
- статус дальше читается по возвращенному `request_id`.
